# Component Folder Structure

| Folder | Purpose | Contents |
|---|---|---|
| `layout/` | Page-level layout primitives | `navigation`, `footer`, `content-wrapper`, `simple-grid` |
| `ui/` | Reusable UI primitives and widgets | `typo`, `icon`, `button`, `tab-group`, `stats-list`, `timeline-as-steps`, `intro-box`, `section-header`, `tool-link`, `cookie-consent`, `faqs` |
| `media/` | Image, video, and logo display | `image-box-container`, `video-box-container`, `logo-carousel`, `logo-carousel-section`, `logo-grid` |
| `sections/` | Reusable page sections | `consultation-cta`, `consultation-cta-default`, `customer/` |
| `heroes/` | Hero banner variants | `hero-2-column`, `background-hero`, `video-background-hero`, `video-background-hero-client` |
| `cards/` | Card components | Card variants |
| `buttons/` | Button components | `contact-button`, `internal-link` |
| `blog/` | Blog-specific components | Blog layout, hero, list, image, separator |
| `contact/` | Contact page components | Contact form and related |
| `team/` | Team page components | Team member display |
| `hiring/` | Hiring components | Job listings and applications |
