# n8n Schulungen (Training & Workshops) - Research

## 1. Types of n8n Training Available

### Beginner Level (Grundlagen)
- **n8n Grundlagen - Einstieg in Workflow-Automatisierung**: Introduction to n8n, system requirements, installation, basic concepts, user interface, creating simple workflows
- **Free n8n Certification Course (Level 1)**: Official beginner course combining theoretical concepts with practical business workflows
- **Beginner Topics**: Introduction to workflows, APIs and Webhooks, understanding nodes, data structures in n8n, core workflow concepts, useful nodes, error handling, debugging, and collaboration

### Advanced Level (Fortgeschrittene)
- **Complex Data Flows**: Advanced technical nodes, conditional logic, branching
- **Enterprise Features**: Sub-workflows, error workflows, pinning and editing output data, handling files
- **Advanced Workflow Techniques**: Data manipulation and transformation, complex workflow design, integration of APIs and third-party services
- **AI-Driven Decision-Making**: Custom AI nodes, AI services integration

### Specialized Training (Spezialisierte Schulungen)

**AI-Agent Automation (KI-Agenten)**
- **Modul 1 - Automatisierung mit KI-Agenten**: Sequential vs. parallel agent execution, coordinating content agents, decision agents and action agents, webhook integration for agent-driven workflows, database integration with intelligent data processing
- **Modul 2 - Multi-Agent-Systeme & MCPs**: Memory systems, multi-agent architectures, MCP-server development
- **AI Integration**: Connecting ChatGPT, Claude, OpenAI, Gemini, and other LLMs to n8n workflows

**IoT Automation**
- Specialized courses for IoT automation scenarios
- Integration with IoT devices and services

**Enterprise Security & Governance**
- Custom project roles and user provisioning via SSO
- Security best practices for enterprise deployments
- User management and security protocols

## 3. Topics Covered in Training Programs (Schulungsinhalte)

### Foundation Topics (Grundlagen)
1. **Introduction to n8n**
   - What is n8n and its development characteristics
   - Comparison to other automation systems (Zapier, Make, RPA tools)
   - Architecture, nodes, triggers, deployment options
   - Real-world use cases

2. **System Setup**
   - System requirements and installation
   - Deployment options: Docker, Proxmox LXC, AWS
   - Self-hosting vs. cloud options
   - Environment configuration

3. **Workflow Basics**
   - Basic concepts and user interface navigation
   - Understanding the Editor UI
   - Creating simple workflows
   - Data flow and structure
   - Node parameters and credentials configuration

4. **Triggers and Execution**
   - Cron triggers (scheduled automation)
   - Webhook triggers (event-driven automation)
   - Manual triggers
   - Understanding the execution engine

### Intermediate Topics (Mittelstufe)
1. **Data Management**
   - How n8n structures data
   - Data manipulation and transformation
   - Data validation and input checking
   - Cleaning and processing data

2. **Integration Capabilities**
   - Integration of APIs and third-party services
   - 500+ pre-built integrations (Google Sheets, email, Slack, HubSpot, etc.)
   - Capturing form and web data
   - Sending automated responses

3. **Logic and Control Flow**
   - Adding structure with If and Switch nodes
   - Conditional branches
   - Error-handling strategies
   - Workflow architecture best practices

### Advanced Topics (Fortgeschrittene)
1. **Complex Workflows**
   - Multi-step workflows
   - Sub-workflows for modular design
   - Error workflows for robust automation
   - Reusable workflow templates

2. **AI and Agent Development**
   - Building AI agents powered by Large Language Models
   - Connecting OpenAI, Claude, ChatGPT, Gemini
   - AI-driven decision-making
   - Custom AI nodes and integrations

3. **Enterprise Features**
   - User management and access control
   - Custom project roles and SSO integration
   - Deployment and scaling strategies
   - High-performance scaling (Redis, RabbitMQ, Docker Swarm)

4. **Best Practices**
   - Modular design principles
   - Breaking down complex processes into reusable components
   - Documentation and team collaboration
   - Performance optimization
   - Security and governance
